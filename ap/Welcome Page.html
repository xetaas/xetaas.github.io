What do Unstable, Stable, In Review, and APWorld Only mean?
APWorld Only: Stable and more or less feature complete. Will only be released as an APWorld.

Merged: Will be in the next version of Archipelago.

In Review: Attempting to get merged into Archipelago main. Usually can be assumed to be stable.

Stable: Should be completely playable without major issues/workarounds. The host probably won't encounter much in the way of issues.
Criteria for stable includes no common generation failures, no common accessibility issues, the client doesn't have issues that may require the host to use /send_location, and other similar ideas.

Unstable: Playable, but there's enough issues that the host should be prepared for problems.

Broken in Main: Not playable in the current version of Archipelago without a modified client and/or APWorld.

In Review games are sorted by PR date, except for major updates to supported games which get put at the top.
The rest are sorted alphabetically, with games in the same series put next to each other in series order.
Note: It is difficult to keep up with when a game should be moved from Unstable to Stable. Please feel free to change the tag from Unstable to Stable if you feel it meets the above description for Stable. Someone else will move it to the proper position on the list after validating it.
